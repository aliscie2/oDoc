# Calendar Quick Start

## 🚀 Get Started

### Connect Google Calendar
```
Click "Connect Google" → Complete OAuth → Grant Permissions
```

### Create Event
```
Click Time Slot → Enter Title → Click "Create"
```

### Set Availability
```
Click "Availabilities" → Add New → Set Days/Times → Save
```

## 📋 Common Tasks

**Create Event**: Click slot → Enter title → Add attendees (optional) → Create
**Edit Event**: Click event → Modify → Update
**Delete Event**: Click event → Delete → Confirm
**Manage Availability**: Toolbar → Availabilities → Add New → Configure → Save

## 🎨 Visual Guide

- **Green** = Available slots
- **Red** = Blocked slots
- **Gray** = Past time
- **Blue** = Google events

## ⚠️ Common Issues

**Token Expired**: Disconnect → Connect → Try again
**Events Not Showing**: Refresh page, check console
**Availability Not Visible**: Verify days/times, check CSS loaded

## 🎯 Pro Tips

- Add attendees for Google Calendar invites
- Set availability to show when you're free
- Works on mobile, tablet, and desktop
- Syncs with Google Calendar automatically
